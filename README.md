Pharmacy Management System
Description

Ce projet est une application de gestion de pharmacie développée en JavaFX et connectée à une base de données MySQL. L’application est accessible selon trois rôles distincts :

Administrateur : Gestion complète des employés, clients et produits, visualisation des ventes et des statistiques.

Employé : Gestion des clients et des produits (ajout, modification, suppression).

Client : Consultation de ses informations et interaction avec le système selon les permissions définies.

L’application inclut un processus de connexion complet, sécurisé et fonctionnel à 100%.

Fonctionnalités

Gestion des produits (ajout, suppression, modification, affichage).

Gestion des clients et employés.

Tableau de bord avec statistiques pour l’administrateur.

Connexion sécurisée et différenciée selon le rôle de l’utilisateur.

Liaison complète avec une base de données MySQL pour la persistance des données.

Difficultés rencontrées

Correction de nombreux bugs mineurs ayant entraîné la perte de temps.

Problèmes liés à la base de données, notamment avec les requêtes SQL et la cohérence des données.

Adaptation du design UML initial : certaines fonctionnalités ont nécessité des classes et processus supplémentaires non prévus initialement.

Limitation du temps pour implémenter toutes les fonctionnalités souhaitées ou trop complexes pour cette première version.

État actuel

L’application fonctionne à 100% pour les fonctionnalités déjà implémentées.

Le projet n’est pas encore complet, certaines fonctionnalités avancées ou complexes restent à développer.

Remarques

Ce projet constitue une première expérience de développement d’une application complète et complexe. Il sert de base solide pour des améliorations futures et une extension des fonctionnalités.
